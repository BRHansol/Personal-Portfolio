# Lab #1 - Network Protocols

## 📋 Overview
Practical laboratory exercise focusing on the fundamental network protocols and their operations using Cisco Packet Tracer.

## 📄 Files
- **[Lab 1.pdf](./Lab%201.pdf)**
- **[ITN_Module3_ProtocolLab.pkt](./ITN_Module3_ProtocolLab.pkt)**

## 🎯 Objectives
- Observe the flow of network traffic and protocol operations
- Analyze PDU (Protocol Data Unit) information at various OSI layers
- Understand the roles of common network protocols in communication

## 🌐 Topics Covered
- Network Protocols (HTTP, TCP, IP, Ethernet)
- Packet analysis in Simulation Mode
- Protocol Data Units (PDUs) and Encapsulation

## ✅ Key Learnings
- Visualized how different protocols work together to exchange data
- Gained hands-on experience inspecting packet details
- Deepened understanding of the OSI and TCP/IP networking models
